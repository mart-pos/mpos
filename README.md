# MPOS Core

Base project for MPOS Core, a local desktop bridge between `martpos.app` and printers connected to the user's machine.

## Requirements

The complete functional, technical, and security requirements are documented in [docs/mpos-core-requirements.md](/Users/jeffersonlopez/Documents/Workspaces/mpos/docs/mpos-core-requirements.md).

## Current Stack

- Tauri 2
- Rust
- React
- TypeScript
- Vite
